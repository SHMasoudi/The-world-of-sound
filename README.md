> fun script for generate piano sound from your code or texts

## Usage
```bash
$ git clone https://github.com/sheyda1997/The-world-of-sound
$ pip3 install playsound
$ pip3 install termcolor
$ cd The-world-of-sound/
$ python piano.py
```
